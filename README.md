# 创锦排课系统

## 项目简介
这是一个基于Electron的桌面应用，开箱即用，用于管理学生课表、教师排班和总课表

## 使用手册
在顶部导航栏→帮助→关于里面可以找到这个仓库的地址
以下所有课表均实现了课时数的自动计算和统计
### 侧边栏
包括学生模块，教师模块，教师总排班，总课表，，自动排课
硬要说的话左下角的profile也算一个模块吧
### 学生模块
#### 搜索框：
侧边栏的搜索框下面可以添加学生，添加后支持模糊搜索
#### 添加学生信息：
需要输入姓名，总课时，开始日期和结束日期（结束日期可以不填）
添加后可以在左侧边栏每个学生名字后面的三个点进行更改
#### 三个点：
三个点还用于设置教师，设置可排课时段和删除学生
1.设置教师从已有的教师中进行选择，**开始排课前必须要为学生分配教师**
2.设置可排课时段会弹出一张空课表，批量拖选和点击选择均可。**注意可排课时段不与周绑定，每周独立的排课时段还在todolist，敬请期待**
3.删除学生，顾名思义。
#### 切换周段：
位于课表上方，可以切换周段
#### 编辑标题：
标题可以双击，改一个自己喜欢的名字，导出png时也是用的这个名字
#### 编辑课表：
按钮在右上角。点击编辑课表，然后单击单元格就可以手动排课了。
包括科目（听力，阅读，口语，写作），类型（可选，如网课、一对一），备注（可选）
#### 导出png：
也在右上角，略
### 教师模块
由于目前老师数量不多，没有设计搜索功能。如果需要请联系我
新增老师只需要输入名字。
#### 三个点：
可以删除教师和查看课时统计。课时统计包括年度统计和月度统计，实时更新
#### 设置休息时段：
双击单元格即可设置，休息时段的单元格会变灰
如果想批量设置，可以点击上方的星期表头（比如Monday），可以批量设置休息时段，包括上午，下午和全天。晚上休息指的是9th和10th，其实属于下午的范围，还没来得及改，todolist
教师排班表与学生课表同步，无需手动更改。可导出png
### 教师总排班
主要反馈教师在各时段的上课和休息情况。可导出png
### 总课表
一周内所有学生的所有排课。可导出png
### 自动排课
2.0.1重磅功能，浩川提的需求。
按照步骤一步步操作即可，我觉得应该没有什么上手难度。
排课方案最多显示15种，如果全列出来我怕页面崩掉。想要更多方案选择的话联系我
### profile
管理员（chuangjin）拥有管理用户、查看系统日志、生成和删除系统日志的权限。
管理用户可以分配新用户，并为用户设置和修改账号密码。
另外三个功能不打职业不用学。
普通账户可以修改自己的头像和密码，聊胜于无。
## 安装和运行

### 开发环境

1. 安装依赖
```bash
yarn install
```

2. 启动开发服务器
```bash
yarn start
```

### 生产打包

```bash
yarn electron:build
```

打包后的安装程序位于 `dist-electron` 目录。

## 项目结构
```
schedule-system/
├── electron/           # Electron主进程
│   ├── main.js        # 入口文件
│   ├── preload.js     # 预加载脚本
│   ├── database/      # 数据库模块
│   ├── services/      # 业务逻辑
│   └── ipc/           # IPC通信
├── src/               # React渲染进程
│   ├── components/    # 组件
│   ├── store/         # 状态管理
│   └── utils/         # 工具函数
└── resources/         # 应用资源
```

## 数据存储

数据存储在用户目录：
- Windows: `C:\Users\{用户名}\AppData\Roaming\创锦排课系统\`
- macOS: `~/Library/Application Support/创锦排课系统/`
- Linux: `~/.config/创锦排课系统/`
